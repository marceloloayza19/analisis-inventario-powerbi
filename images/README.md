# Capturas del dashboard

Esta carpeta debe contener las capturas exportadas desde Power BI Desktop cuando el dashboard sea construido manualmente.

## Capturas recomendadas

- `dashboard_general.png`: vista completa del dashboard.
- `kpis_principales.png`: tarjetas de Stock Total, Valor Total de Inventario, Productos Criticos y Unidades Vendidas.
- `productos_criticos.png`: grafico de productos con stock critico.
- `valor_inventario_categoria.png`: valor de inventario por categoria.
- `productos_mas_vendidos.png`: top 10 productos mas vendidos.
- `distribucion_estado_stock.png`: grafico de dona por estado de stock.
- `detalle_inventario.png`: tabla o matriz con el detalle de inventario.

## Recomendaciones

- Guardar las capturas en formato PNG.
- Usar nombres claros y consistentes.
- Verificar que los textos sean legibles.
- No agregar capturas hasta que el dashboard real este construido.
